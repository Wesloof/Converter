Программа предназначена для изменения параметров датчика в .ini файле. С помощью математических формул происходит конвертация необходимых значений для двух типов датчиков: на 45 ом и на 0.5 ом.
Программа обрабатывает значения в файле, изменяет их на нужные, далее заменяет их уже на сконвертированные значения, перенося также неизмененные значения.
